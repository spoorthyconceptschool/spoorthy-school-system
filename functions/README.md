# Firebase Cloud Functions

This directory will contain the server-side logic for the application.

## Structure
- `src/index.ts`: Entry point.
- `src/services/authService.ts`: Role management.
- `src/services/paymentService.ts`: Razorpay integration.
- `src/services/studentService.ts`: Student logic.

To initialize:
Run `firebase init functions` and select TypeScript.
